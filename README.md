# KitaSehat
KitaSehat merupakan aplikasi yang dibangun untuk memenuhi tugas akhir. Aplikasi ini dikembangkan sesuai dengan modul pendaftaraan,admisi, dan antrean pada rumah sakit. KitaSehat dikerjakan oleh dua mahasiswa yaitu Dwiki Cahyo Pambudi (Android Developer) dan Ariasatya Mahatma (Backend Developer).

**Login**

<img src="https://github.com/DwikiCahyo/KitaSehat-TA-/assets/70952085/2920f30c-94ea-4a96-aa55-178318032d9b"  width="300" height="600">

Halaman Login merupakan Halaman Awal untuk Pengguna masuk ke dalama aplikasis, Pengguna hanya perlu memasukan NIK dan password untuk dapat masuk kedalaam aplikasi.

**Registrasi Data Pasien**

<img src="https://github.com/DwikiCahyo/KitaSehat-TA-/assets/70952085/06521e6a-c92c-41b1-81e7-a689868c4b0f"  width="300" height="600">
<img src="https://github.com/DwikiCahyo/KitaSehat-TA-/assets/70952085/07092f5e-5308-4393-81c6-32b2bce445c8"  width="300" height="600">

Halaman registrasi dibuatkan berdasarkan akan kebutuhan pengguna untuk melakukan input pada data pribadi mereka. Terdapat dua jenis data yang digunakan untuk proses registrasi yaitu Data Pribadi dan Data Sosial. Sebelum melakukan registrasi penggunaa juga dipastikan untuk menyetujui penggunaan data mereka ketika ingin menggunkan aplikasi.

**Registrasi Akun Pasien**

<img src="https://github.com/DwikiCahyo/KitaSehat-TA-/assets/70952085/78588649-7921-4251-bf79-b047e2faad23"  width="300" height="600">

Halaman registrasi akun pasien dikhususkan untuk proses pembuatan akun. Data yang diperlukan yaitu NIK dan Password dengan validasi aplikasi yang telah dibuat.

**Dashboard**

<img src="https://github.com/DwikiCahyo/KitaSehat-TA-/assets/70952085/7c51abcd-ecff-49e3-9f1c-815d5ac0f86d"  width="300" height="600">

Halaman Dashboard berisi nomor antrean pasien dan fitur fitur yang tersedia pada aplikasi. Halaman ini kan menampilkan nomor pasien dan detail singkat dari antrean yang dimiliki oleh pasien ketika sudah mendaftar kedalam antrean.

**Pendaftaraan Antrean Layanan**

<img src="https://github.com/DwikiCahyo/KitaSehat-TA-/assets/70952085/0d337251-afb0-4185-bc36-59cd355386b3"  width="300" height="600">
<img src="https://github.com/DwikiCahyo/KitaSehat-TA-/assets/70952085/1cbb242f-2b65-4a71-a4b3-6b607158644c"  width="300" height="600">

Proses pendaftaraan antrean diawali dengan memilih rumah sakit tersedia, setelah memilih rumah sakit pengguna akan diarahkan untuk mengisis keluhan seerta memilih fasilitas yang ingin dituju. Pemilihan layanan dapat melaului layanan umum maupun layanan bpjs. Jika melalui layanan BPJS nomor BPJS akan terisi sesuai deng yaang telah didaftarkan.

**Informasi Rumah Sakit**

<img src="https://github.com/DwikiCahyo/KitaSehat-TA-/assets/70952085/5bb5ba49-ceab-4a90-a487-780f2c96da5d"  width="300" height="600">

Pada halaman informasi rumah sakit akan menampilkan peta dan list rumah ssakit yang menunjukan rumah sakit yang tersedia disekitar kita, Peta tersebut juga berfungsi untuk menunjukan lokasi dan rute dari rumah sakit tersebut dengan bantuan Google Maps. Ketika pengguna menekan tombol detail dari list rumah sakit makan menaampilkan detail dari rumah sakit dan informasi dari fasiltas yang tersedia.

- Google Map Service
